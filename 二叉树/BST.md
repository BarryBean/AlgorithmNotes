# 特点

BST 二叉搜索树：

- 根节点 >= 左子树所有节点；
- 根节点 <= 右子树所有节点；
- 左右子树也是 BST。

BST 的特点是中序遍历有序。

一般 BST 题目从定义和特点两个角度出发解决。
